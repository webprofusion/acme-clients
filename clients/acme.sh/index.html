acme.sh

Popular acme client written as unix shell script. Unlike many other popular clients (which tend to default to using Let's Encrypt), acme.sh defaults to the ZeroSSL certificate authority for certificate orders.

Pros

  • Optimized for Linux and other unix-like systems, broad compatibility
  • Minimal dependencies
  • Support for a range of DNS APIs (external dependencies required).

Cons

  • Only limited support for Windows (requires Cygwin or other compatible shells etc)
  • Lacks a dedicated support community
  • No automated update notifications

Features

  • Option to use a custom Certificate Signing Request (CSR) generated from an external source.
