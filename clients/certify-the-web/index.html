Certify The Web

Professional Automated Certificate Management UI for Windows servers, including direct support for IIS and a wide range of deployment tasks.

Pros

  • Popular choice on Windows, with hundreds of thousands of users.
  • Optimised for Windows users, with built-in support for IIS and the windows certificate store.
  • Registered users have access to dedicated helpdesk ticketing.
  • Support for a wide range of DNS APIs (28+, including many provided via Posh-ACME).
  • Support multiple auth config (e.g. mixing http and DNS validation, or using multiple DNS providers in one cert).
  • Extensive range of optional Deployment Tasks to perform scripting or to deploy to Apache, nginx, Azure Key Vault etc.

Cons

  • Currently only available on Windows.
  • Commercial users require a license after their evaluation.

Features

  • HTTP domain validation.
  • DNS domain validation.
  • Option to combine multiple domain validation configurations when working with a single certificate.
  • Option to use a different certificate authority per certificate.
  • Option to use a custom Certificate Signing Request (CSR) generated from an external source.
  • Supports automatic fallback to a different CA if renewals fail with default/selected CA.
  • Supports ACME ARI (draft) extensions for renewal info checking.
  • Supports ACME Profiles (draft) extension for certificate feature selection.
